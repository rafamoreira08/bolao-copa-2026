🔑 Cadastro de Palpites

Página do administrador — não compartilhe este link

Como funciona: preencha os palpites abaixo (deixe em branco quem não enviou), clique em Gerar arquivo, copie o conteúdo e cole no GitHub substituindo o arquivo dados.js. As duas páginas do bolão atualizam sozinhas em ~1 minuto.

🏁 Resultado final: quando um jogo terminar, preencha também o resultado final dele — isso "congela" a pontuação, pois a API da ESPN às vezes esquece placares de dias anteriores.
Depois de copiar:
1. Abra o editor do dados.js no GitHub
2. Selecione tudo (Cmd+A) e cole por cima (Cmd+V)
3. Clique em Commit changes — pronto, no ar em ~1 minuto ⚽
